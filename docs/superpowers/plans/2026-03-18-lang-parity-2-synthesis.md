# Language Parity Plan 2: Test Synthesis Expansion

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add `TestSynthesizer` implementations for all 8 languages currently missing synthesis: Go, C++, C, C#, Swift, Kotlin, Ruby, WASM.

**Architecture:** Each synthesizer follows the exact same pattern as `PytestSynthesizer` — a struct with `output_dir`, implementing `TestSynthesizer` trait with `language()` and `synthesize()`. The only language-specific parts are: file extension, header/imports, chunk size, and code wrapping format. All share the same dedup + chunk + hash + write logic.

**Tech Stack:** Each language's native test framework (go test, gtest, xunit, XCTest, minitest/rspec, etc.)

**Depends on:** Plan 1 (Ruby sandbox) for Ruby to be testable.

- [ ] **Step 3: Implement GoTestSynthesizer**

Copy the `PytestSynthesizer` pattern. Key differences:
- File extension: `.go`, named `apex_generated_<hash>_test.go`
- Header: `package apex_generated\n\nimport "testing"\n\n`
- Comment prefix: `//` not `#`
- Chunk size: 20
- No indentation wrapping (Go tests are top-level functions)

```rust
use apex_core::{
    error::{ApexError, Result},
    traits::TestSynthesizer,
    types::{Language, SynthesizedTest, TestCandidate},
};
use std::{collections::HashSet, path::PathBuf};
use tracing::info;

pub struct GoTestSynthesizer {
    output_dir: PathBuf,
}

impl GoTestSynthesizer {
    pub fn new(output_dir: impl Into<PathBuf>) -> Self {
        Self { output_dir: output_dir.into() }
    }

    fn generate_file_content(candidates: &[&TestCandidate]) -> String {
        let mut content = String::new();
        content.push_str("// Auto-generated by APEX — do not edit by hand.\n");
        content.push_str("package apex_generated\n\n");
        content.push_str("import \"testing\"\n\n");

        for (i, candidate) in candidates.iter().enumerate() {
            if !candidate.code.trim().is_empty() {
                content.push_str(&format!(
                    "// Candidate {} — targets: {}\n",
                    i + 1,
                    candidate.target_branches.iter()
                        .map(|b| format!("{}:{}", b.file_id, b.line))
                        .collect::<Vec<_>>().join(", ")
                ));
                content.push_str(&candidate.code);
                content.push_str("\n\n");
            }
        }
        content
    }
}

impl TestSynthesizer for GoTestSynthesizer {
    fn language(&self) -> Language { Language::Go }

    fn synthesize(&self, candidates: &[TestCandidate]) -> Result<Vec<SynthesizedTest>> {
        if candidates.is_empty() { return Ok(Vec::new()); }

        let mut seen: HashSet<u64> = HashSet::new();
        let mut unique: Vec<&TestCandidate> = Vec::new();
        for c in candidates {
            use std::hash::{Hash, Hasher};
            let mut h = std::collections::hash_map::DefaultHasher::new();
            c.code.hash(&mut h);
            if seen.insert(h.finish()) { unique.push(c); }
        }

        let mut outputs = Vec::new();
        for chunk in unique.chunks(20) {
            let content = Self::generate_file_content(chunk);
            let hash: u64 = {
                use std::hash::{Hash, Hasher};
                let mut h = std::collections::hash_map::DefaultHasher::new();
                content.hash(&mut h);
                h.finish()
            };
            let file_name = format!("apex_generated_{hash:016x}_test.go");
            let file_path = self.output_dir.join("apex_generated").join(&file_name);

            let parent = file_path.parent()
                .ok_or_else(|| ApexError::Synthesis("no parent dir".into()))?;
            std::fs::create_dir_all(parent)
                .map_err(|e| ApexError::Synthesis(format!("create dir: {e}")))?;
            std::fs::write(&file_path, &content)
                .map_err(|e| ApexError::Synthesis(format!("write: {e}")))?;

            let covers = chunk.iter().flat_map(|c| c.target_branches.clone()).collect();
            info!(file = %file_path.display(), "wrote Go test file");
            outputs.push(SynthesizedTest { path: file_path, content, covers_branches: covers });
        }
        Ok(outputs)
    }
}
```

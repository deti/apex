# Language Parity Plan 4: Fuzzing Expansion

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add fuzzing support for C# and Swift — the only two languages currently missing it. The generic byte-level fuzzer already works; this plan adds language-aware mutators and harness generation.

**Architecture:** The `FuzzStrategy` in `apex-fuzz` is language-agnostic (operates on `Vec<u8>`). Language-specific support means: (1) a mutator that understands the language's input format, (2) a harness generator that creates compilable fuzz targets, (3) wiring in the CLI. The existing `MOptScheduler` handles mutator selection automatically.

**Tech Stack:** LibAFL (optional), .NET fuzzing via SharpFuzz, Swift fuzzing via swift-fuzz

**Depends on:** Nothing — independent of Plans 1-3.

/// Generate a SharpFuzz harness for a C# method.
pub fn csharp_fuzz_harness(class: &str, method: &str, params: &[&str]) -> String {
    let uses_stream = params.iter().any(|p| p.contains("Stream"));
    let mut harness = String::new();
    harness.push_str("// Auto-generated by APEX\n");
    harness.push_str("using SharpFuzz;\nusing System;\nusing System.IO;\n\n");
    harness.push_str(&format!("namespace ApexFuzz\n{{\n"));
    harness.push_str("    class Program\n    {\n");
    harness.push_str("        static void Main(string[] args)\n        {\n");

    if uses_stream {
        harness.push_str(&format!(
            "            Fuzzer.Run(stream => {{\n                {class}.{method}(stream);\n            }});\n"
        ));
    } else {
        harness.push_str(&format!(
            "            Fuzzer.Run(data => {{\n                var input = System.Text.Encoding.UTF8.GetString(data);\n                {class}.{method}(input);\n            }});\n"
        ));
    }

    harness.push_str("        }\n    }\n}\n");
    harness
}
```

/// Generate a swift-fuzz harness for a Swift function.
pub fn swift_fuzz_harness(type_name: &str, method: &str, params: &[&str]) -> String {
    let mut harness = String::new();
    harness.push_str("// Auto-generated by APEX\n");
    harness.push_str("import Foundation\n\n");
    harness.push_str("@_cdecl(\"LLVMFuzzerTestOneInput\")\n");
    harness.push_str("func fuzzTest(_ data: UnsafePointer<UInt8>, _ size: Int) -> Int32 {\n");
    harness.push_str("    let bytes = Data(bytes: data, count: size)\n");
    harness.push_str(&format!("    let _ = try? {type_name}.{method}(bytes)\n"));
    harness.push_str("    return 0\n");
    harness.push_str("}\n");
    harness
}
```
